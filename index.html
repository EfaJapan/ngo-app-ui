🏗 リレーション設計図
ドナー台帳 ── N:1 ──▶ プロジェクト台帳 ◀── 1:N ── 送金台帳
▸ 送金台帳 → 国・年度はプロジェクトからロールアップ取得 ▸ 不足額 = 目標調達額合計 − 確定資金額 ▸ 送金は四半期×国で集計
① プロジェクト・管理費管理台帳
フィールド型属性説明
project_idUUIDPK主キー（自動）
nameTEXT必須プロジェクト・管理費名
countryENUM必須ラオス / カンボジア / ベトナム（拡張可）
fiscal_yearINTEGER必須年度（2025〜2031）
target_amountBIGINT必須目標調達額合計（円）
confirmed_fundsBIGINT—確定資金額（円）
shortage [AUTO]BIGINT—target_amount − confirmed_funds
fund_status [AUTO]ENUM—shortage＞0→資金不足 / else→安定
② 助成金・ドナー管理台帳
フィールド型説明
donor_idUUID主キー
donor_nameTEXTドナー・助成機関名
project_id → FKRELATION対象プロジェクト
priorityENUMA / B / C
recruit_periodTEXT公募期間
decision_timingTEXT決定時期
applied_amountBIGINT申請額（円）
adopted_amountBIGINT採択額（円）
adoption_rate [AUTO]DECIMALadopted/applied×100（%）
resultENUM申請中 / 採択 / 不採択
apply_dateDATE申請した日付
result_dateDATE結果通知日付
payment_confirm_dateDATE入金確認日
expected_paymentYEARMONTH入金予定時期
urlTEXT助成申請情報URL
notesTEXT備考（自由記入）
③ 海外送金管理台帳（四半期×国別集計）
フィールド型説明
remit_idUUID主キー
project_id → FKRELATION対象プロジェクト
country [ROLLUP]TEXTプロジェクトから自動取得
fiscal_year [ROLLUP]INTEGERプロジェクトから自動取得
quarterENUMQ1(4-6月)/Q2(7-9月)/Q3(10-12月)/Q4(1-3月)
planned_amountBIGINT計画送金額（円）
actual_amountBIGINT実績送金額（円）
remit_dateDATE送金日
execution_rate [AUTO]DECIMALactual/planned×100（%）
notesTEXT備考
